Listado de Órdenes

Se verán todas las órdenes guardadas en el sistema, activas y terminadas.
Se verán todos los detalles de cada orden.
Se accesa la página de Detalle de Orden al hacer click en el número de Orden (primer columna).

Detalle de Orden específica

Se podrán ver todos los procesos que una orden lleva.
Se podrá comenzar/terminar un proceso.
Para comenzar o terminar un proceso se deben proveer el usuario y contraseña de quien está encargado de hacer tal proceso.
El trabajador que comienze un proceso debe terminarlo. Un trabajador no podrá terminar un proceso a menos que lo haya comenzado.
El usuario y contraseña de cada trabajador será creado por el administrador.

Listado de Órdenes Activas

El listado se actualiza automáticamente cada minuto.
El estado de los procesos se actualizará aún sin una conexión a Internet.
El listado de nuevas órdenes activas y órdenes terminadas no se actualizará a menos que haya una conexión a Internet.
